# 👤 User Management System

A simple and interactive **User Management System** built using vanilla JavaScript. It allows users to add, edit, delete, and search user profiles with persistent storage using **localStorage**.

---

## 🚀 Features

* ➕ Add new users with name, role, image, and bio
* ✏️ Edit existing user details
* ❌ Delete users
* 🔍 Search users with debounce optimization
* 💾 Data persistence using localStorage
* 🌙 Light/Dark theme toggle
* 🔔 Toast notifications for actions (add/edit/delete)
* 📱 Responsive and dynamic UI rendering

---

## ⚙️ How It Works

* Users are stored in **localStorage**
* UI is dynamically rendered using JavaScript DOM manipulation
* Form validation ensures correct input before submission
* Event delegation is used for handling edit/delete actions
* Debouncing improves performance during search
